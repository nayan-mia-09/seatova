# ⚽ Seatova (Work in Progress)

Seatova is a football ticket booking web application that I am currently building.
The goal of this project is to create a clean and user-friendly platform where users can explore matches and book tickets easily.

> 🚧 This project is still under development — features and UI are not final.

---

## 🧠 Current Progress

* Basic UI layout completed
* Match / product data rendering
* Components structure setup
* Routing with React Router
* Some reusable UI components created

---

## 🚀 Features (Implemented So Far)

* View football matches / items
* Display team information (image, play style, etc.)
* Basic card-based UI design
* Navigation between pages
* Responsive layout (in progress)

---

## 🛠️ Tech Stack

* React.js
* Tailwind CSS
* React Router
* JavaScript (ES6)

---

## 📁 Project Structure (Current)

```
src/
│── assets/        # images & static files
│── components/    # reusable UI components
│── pages/         # page-level components
│── data/          # static data (matches/products)
│── App.jsx
│── main.jsx
```

---

## ⚙️ Getting Started

Clone the repository:

```bash
git clone https://github.com/nayan-mia-09/seatova.git
cd seatova
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
